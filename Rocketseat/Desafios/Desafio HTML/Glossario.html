Elementos HTML

Títulos e parágrafos
Há 6 tipos diferentes de tags para títulos sendo o <h1> o título principal, indo até o <h6> como os subtítulos.
A tag <p> é a tag para separação de parágrafos.
Listas
As listas podem ser ordenadas ou não ordenadas, a tag para lista é a <li>.
OBS: li(vezes o número de itens na lista adianta a quantidade da lista) ex li*5. Exemplos:
Lista ordenada <ol>(aqui entra as <li>) </ol>
Lista não ordenada <ul>(aqui entra as <li>)</ul>.
Detalhes de contato
Tag <address> é uma tag para adicionar o contato ao final da página. Definindo quem compilou e aonde.
